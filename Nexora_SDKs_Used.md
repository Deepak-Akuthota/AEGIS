**SDKs \& Libraries Used:**

1\. whatsapp-web.js SDK: This is the core open-source library used to intercept and automate WhatsApp Web directly from the backend.

Link -  https://wwebjs.dev/

2\. Socket.io SDK: Used for the real-time bi-directional link between the Backend AI and the Frontend React dashboard.

Link -  https://socket.io/

3\. Framer Motion SDK: The animation library used to create the smooth Zen transitions and ripple effects.

Link -  https://www.framer.com/motion/

4\. Lucide React: The icon SDK for our premium UI components.

Link -  https://lucide.dev/

5\. discord.js SDK: Included in the backend to manage Discord notification interception.

Link - https://discord.js.org/
